<h1 align="center">Sudoku.click</h1>
<p align="center">
  <b>Ionic and capacitor sudoku implementation for Android and iOS platform</b>
</p>
<br>

## Usage

###Global installation:
```bash
npm install -g @ionic/cli
```

### Project installation:
```bash
npm install
ng serve --port 1338
```
